### :one: K-Nearest Neighbor (KNN)
- ### 1.1 KNN 算法描述
```
KNN算法中K指的是寻找距离测试样本最近的训练集元素个数。

KNN算法描述如下：

1、初始化训练集和类别；

2、计算测试集样本与训练集样本的距离；

3、根据距离大小对训练集样本进行升序排序；

4、选取距离最小的前K个训练样本，统计其在各类别中的频率；

5、返回频率最大的类别，即测试集样本属于该类别。
```
